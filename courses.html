Java Programming

Working in the field of information technology as a programmer requires a firm understanding of Object-Oriented Programming (OOP) concepts. Students explore object-oriented programming methodology using the Java programming language.

Linux

SQL Databases

Databases are used to store data and are a core component of many information technology systems. Students learn the fundamentals of relational database design using Entity Relationship Diagrams (ERDs), and use SQL to create, modify and query a database.

Web Programming

The World Wide Web (WWW) has become an integrated part of everyday life. Students develop basic skills of web programming, website design, and implementation.
